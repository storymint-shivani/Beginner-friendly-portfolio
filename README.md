# dev.port — Personal Portfolio

A dark-themed, single-file personal portfolio built with HTML, CSS & vanilla JavaScript. Designed to showcase projects, skills, and contact info with smooth animations and an interactive UI.

## ✨ Features

- Animated particle background with canvas
- Custom cursor with hover effects
- Typed role animation
- Scroll-triggered reveal animations
- Skill bars with percentage indicators
- Responsive layout (mobile-friendly)
- Active nav link highlighting

## 🛠️ Built With

`HTML5` `CSS3` `JavaScript` `Font Awesome` `Google Fonts`

## 🚀 Getting Started

Just open `index.html` in your browser — no build tools or dependencies needed.

```bash
git clone https://github.com/shivaniranaut/web-portfolio
cd web-portfolio
open index.html
```

## 📁 Structure

```
web-portfolio/
└── index.html   # Everything in one file
```
